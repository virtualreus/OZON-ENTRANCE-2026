## OZON Тестовое задание

Ответы на возможные вопросы и пояснения:
1) Почему именно такая валидация? Почему не использовал тот же validator. - Потому что в рамках проекта у нас достаточно мало проверок (корректность URL - использую регулярное выражение), а на короткую ссылку в целом проверка одного инварианта -> сложных структур в сервис не приходит, поэтому затаскивать ради двух маленьких проверок целый валидатор мне показалось оверхедом - он тут не нужен.
